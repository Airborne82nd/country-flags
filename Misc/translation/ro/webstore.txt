Suplimentul „Country Flags & IP Whois” este o extensie care afișează în bara de unelte un buton cu drapelul statului/țării/teritoriului unde este localizat fizic serverul web curent, folosind o bază de date offline.  Cunoașterea locației serverului adaugă un nivel suplimentar de conștientizare privind navigarea pe internet și poate fi utilă prin indicarea limbilor native și a jurisdicțiilor legale care ar putea fi aplicate.  Informații suplimentare despre geolocalizare sau informații despre Whois pot fi obținute printr-un clic pe pictograma-drapel din bara de unelte.

Pentru întrebări frecvente (FAQ) și rapoarte generale despre erori, vizitați:
-> http://add0n.com/country-flags.html
Pentru rapoartele despre erori tehnice folosiți:
-> https://github.com/andy-portmen/country-flags/issues

Informații despre bazele de date:
[IPv4] https://geolite.maxmind.com/download/geoip/database/GeoIPCountryCSV.zip
[IPv6] https://geolite.maxmind.com/download/geoip/database/GeoIPv6.csv.gz
Rețineți că bazele de date CSV sunt comprimate. Pentru mai multe informații, vizitați pagina de întrebări frecvente (FAQ) și vizualizați codul sursă.

Instrumente disponibile în meniul contextual afișat cu clic-dreapta:
<ul>
<li>Verificare SSL: Verifică certificatul SSL</li>
<li>TraceRoute: Afișează întârzierile pe traseu și de tranzit ale pachetelor</li>
<li>Ping: Testează accesibilitatea adresei IP</li>
<li>TinyURL: Scurtează adresa URL utilizând TinyURL.com</li>
<li>Căutare DNS: Efectuează o căutare DNS autoritară</li>
<li>Căutare Whois: Caută înregistrarea și delegarea numelui de domeniu</li>
<li>Anteturi HTTP: Listează toate anteturile HTTP de răspuns</li>
<li>Alexa: Analizează traficul web, generează statistici și analize</li>
<li>Fișe WOT: Evaluează reputația saitului</li>
<li>Virustotal: Scanează gratuit de viruși și malware adresa URL</li>
<li>Stare server: Verifică dacă saitul este activ sau nu</li>
<li>Google Translate: Traduce instantaneu pagina web</li>
<li>Google Cache: Afișează conținutul paginii din memoria cache de la Google</li>
<li>Wikipedia: Caută informații despre domeniu</li>
<li>intoDNS: Verifică starea și configurația domeniului și furnizează rapoarte DNS și ale serverelor de mail</li>
<li>Netcraft: Furnizează informații privind securitatea saitului</li>
<li>Wave Report: Furnizează informații despre accesibilitate și stil și erorile din pagina web</li>
<li>Validate.nu: Validează conținutului HTML5 al paginii</li>
<li>Validator CSS W3C: Validează stilurile CSS ale paginii</li>
<li>Validator W3C: Validează conținutului HTML5 al paginii</li>
<li>Arhivă Internet: Accesează arhiva universală a paginii web</li>
<li>Google: Caută pe Google după domeniu</li>
<li>Wolfram|Alpha: Furnizează diferite date și calcule despre domeniu</li>
<li>Copiere IP: Copiază adresa IP în clipboard</li>
</ul>

Comenzile personalizate acceptă următorii parametri:
<ul>
<li><code>[lang]</code>: codul limbii navigatorului (ex.: en-US, ro, de)</li>
<li><code>[url]</code>: adresa URL completă</li>
<li><code>[enurl]</code>: adresa URL codată cu <i>encodeURIComponent()</i></li>
<li><code>[host]</code>: numele serverului gazdă</li>
<li><code>[curl]</code>: adresa URL fără parametri</li>
<li><code>[ip]</code>: adresa IP a serverului</li>
</ul>

Începând cu versiunea 0.2.0, este posibilă deschiderea filelor noi în fundal și/sau alături de fila curentă.
